Javascript

O que é Javascript

JavaScript é uma linguagem de programação de alto nível criada, a princípio, para ser executada em navegadores e manipular comportamentos de páginas web. Com seus scripts é possível incluir, em uma página estática, elementos dinâmicos como mapas, formulários, operações numéricas, animações, infográficos interativos e muito mais.

O JavaScript é uma das mais importantes tecnologias voltadas para o front-end e, unindo-se ao trio HTML, CSS e PHP, formam um grupo de linguagens que abrangem praticamente todas as exigências do desenvolvimento de uma página completa, dinâmica e com boa performance.
Mas o JS não se restringe mais apenas às páginas e aos navegadores, como foi durante vários anos: com o advento de diversos frameworks, APIs, melhorias e criação de centenas de funções, hoje já é possível utilizar JavaScript em aplicativos mobile, softwares para desktop e até mesmo em back-end.

JavaScript, originalmente, é uma linguagem de programação client-side, ou seja, é executada do lado do usuário, mais especificamente pelo navegador utilizado por este usuário. Em outras palavras, isso significa que todas as suas ações são processadas na máquina de quem as utiliza, sem a necessidade de enviá-las a nenhum outro ambiente. Como nada é enviado a nenhum servidor externo para processamento, as respostas são imediatas.

Onde colocar o codigo javascript?

Existem 2 formas principais de colocar o codigo javascript em uma pagina.


Primeira: abrindo em qualquer parte do html uma tag < "script" > e colocando a linha do codigo a ser executado dentro dela < /script >


Segunda: fazendo um arquivo externo igual com o css, ex script.js e importando ele no head ou no body através da tag < scr ipt src="script.js"> < /scri pt>


É possivel definir a ordem em que o javascript da pagina sera carregado, colocando ele dentro do head, dessa forma o javascript é carregado antes mesmo do body do site, ou colocando ele no final do body, e dessa forma ele é executado e aplicado nos conteudos da pagina após o carregamento do body. É recomendado que se siga essa segunda forma.

Console

O console é uma area em que é permitida aplicações de javascript a todo o navegador. Ele registra informações como solicitações de rede, JavaScript, CSS, erros e avisos de segurança, além das mensagens explicitas registradas pelo código JavaScript.

Variaveis

Variáveis em Javascript são uma forma para se salvar dados que possam ser acessados e/ou manipulados posteriormente, independente do tipo desses dados, strings, números, booleanos, arrays, objetos ou funções.


Como criar variaveis:


var nomeDaVariavel = "valorDaVariavel";

var nome = "Andressa";

var idade = 24;


Importante!: O nome da variavel não pode conter espaço, caractere especial e nem começar com numero. Além disso, o valor da variavel se for texto deve-se colocar entre "", e se for numero não deve ter aspas.


Concatenação: consiste na junção de variaveis. Ex:


var nome = "Andressa";

var sobrenome = "Martins";

var nomeCompleto = nome + " " + sobrenome;


console.log(nomeCompleto) vai ser igual a Andressa Martins


var primeiraNota = 9;

var segundaNota = 10;

var somaDasNotas = primeiraNota + segundaNota;


console.log(somaDasNotas) vai ser igual a 19


Para se adicionar um espaço entre uma variavel e outra é utilizado um espaço entre aspas e incluir na soma " ".


Somente são realizadas as expressões matematicas ( + - / * ) se os valores das variaveis estiverem escritos sem o uso das aspas, caso contrario, quando o numero é colocado dentro de aspas, ele é interpretado como um texto.

Tipos de definições de variaveis

Var:


Ex: var nome = "Andressa";


Nesse tipo de definição, a variavel é enviada para o escopo geral da tela, então é possivel usa-la em variaveis globais como window. É possivel ter acesso a ela em qualquer area do sistema.

Essa variavel pode ser sobreescrita após uma redefinição de seu valor. Dessa forma, o valor é sempre o ultimo que foi definido.


Por exemplo:

var nome = "Andressa";
var nome = "Willian";
var nome = "Gabriel";
var nome = "Zoe";

Dessa forma, o valor final da variavel é "Zoe", pois foi o ultimo valor definido.

Let:


Ex: let nome = "Andressa";


Nesse tipo de definição, a variavel fica disponivel apenas em um escopo espefifico do codigo. Apenas em uma area especifica do codigo.


Por exemplo:

let nome = "Andressa";
if (nome == "Andressa") {
let idade = 90;
console.log(idade);
}

Nesse exemplo, variavel (idade) só funciona se estiver dentro da "caixa de condição" {}, se tentar usa-la fora da condição, colocando o console.log(idade) fora da caixa de codigo, por exemplo, não vai ser possivel identificar a atribuição dessa variavel.


O ponto positivo do let é que evita o consumo desnecessario da memoria. E evita confusão entre as variaveis ao longo do codigo.

Outro ponto importante é que não é possivel redefinir o valor da variavel let em uma mesma area de codigo.


Por exemplo:

let nome = "Andressa";
let nome = "Willian";
Essa forma esta incorreta.


Const:


Ex: const nome = "Andressa";


Nesse tipo de definição a variavel é definida com um valor constante, ou seja, é impossivel modificar o valor dessa variavel. Uma vez definido o seu valor, este será o unico possivel dentro do codigo inteiro.

Condicionais - If e Else

A estrutura condicional if permite ao JavaScript executar um trecho de código somente se uma determinada condição for verdadeira.


Por exemplo:

var hora = 9;
if ( hora < 12 ) {
console.log("Bom dia");
} else {
console.log("Não é de manhã");
}


Nesse exemplo, foi definifida a seguinte condição: se a hora for menor que 12, então o resultado da informação será "bom dia", caso não se cumpra essa condição, então o resultado é "não é de manha"


var hora = 9;
if ( hora < 12 ) {
console.log("Bom dia");
} else if ( hora < 18 ) {
console.log("Boa tarde");
} else if ( hora <= 23 ) {
console.log("Boa noite");
}

Ja nesse exemplo a condição foi a seguinte: Se a hora for menor que 12, então o resultado é "bom dia", caso essa condição não se cumpra então verificar se a hora é menor do que 18, se positivo, o resultado é "boa tarde", caso essa condição também não seja cumprida, então verificar se a hora é menor ou igual a 23, se positivo o resultado é "boa noite".


var hora = 12;
if ( hora >= 12 && hora < 18 ) {
console.log("Boa tarde");
}

Nesse exemplo se utilizou uma condição com dois valores a serem verificados. Se hora é maior ou igual a 12 e se a hora for menor do que 18, então o resultado é "boa tarde"


var hora = 18
if ( hora == 12 || hora == 18 ) {
console.log("Voce esta na hora do rush");
}

Nesse ultimo exemplo a condição foi a seguinte: se a hora for igual a 12 ou 18, então o resultado final é "você está na hora do rush".
Importante: nas condicionais, o "igual" é representado por == e o "ou" é representado por ||

DOM

Document Object Model - É a referencia aos elementos da estrutura da pagina/documento para serem manipulados via javascript.

Selecionando Elementos

document.getElementById


document.getElementById('exemplo').innerHTML = "< button> Botão< /button>"


Ou seja, Pegar o elemento do documento que tem o id "exemplo" e dentro do HTML fazer a seguinte modificação = < button>Botão< /button> (Colocar um botão).


getElementsByClassName('exemplo')


Ou seja, pegar os elementos pelo nome da classe. Como é possivel atribuir uma classe a varios elementos, dessa forma é possivel pegar varios elementos com o mesmo nome de classe.


getElementsByClassName('exemplo')[0].innerHTML = "alteracao"


Ou seja, pegar os elementos pelo nome da classe 'exemplo' e dentro do primeiro item '0' (array 0), fazer a seguinte modificação no HTML = alteracao


getElementsByTagName('div')


Ou seja, pegar todos os elementos pelo nome das tags.


getElementsByName('email')

Ou seja, pegar oe elementos pelo nome. ex: names de inputs


Selecionando com querySelector

Através desse comando é possivel fazer uma seleção por id, class, tag, nome, etc. Para retornar todos os resultados, e não somente o primeiro, é possivel utilizar a variação querySelectorAllVeja exemplos:


document.querySelector('#exemplo') - Para selecionar um elemento por id


document.querySelector('.exemplo') - Para selecionar elementos pela classe


document.querySelector('button') - Para selecionar elementos pela tag


document.querySelectorAll('exemplo') - Para selecionar todos os resultados das classes com nome 'exemplo'.
