Html Semântico

por Andressa Martins

No Html semântico, utilizamos tags especificas para semantica e acessibilidade. Essas tags são interpretadas pelos monitores de acessibiidade e são muito valorizadas pelo motor de busca do google, de forma que é muito importante tê-las para a otimização do SEO da pagina. Elas não possuem alterações visuais, mas podem funcionar como divs, sendo possivel inclusive a aplicação de efeitos no css.

Tag Header, Footer e Nav

Tag < header>

Tem função de informar ao navegador que trata-se de uma pagina escrita em html5 para que ele possa traduzi-la


< html>< /html>

Define que se trata de um documento html


< head>< /head>

Dentro dessa tag se escreve algumas configurações da pagina. Como < meta charset="UTF-8"/> , < title> , < link rel="stylesheet" href="arquivo.css"/> veja a definição dessas tags abaixo:


< meta charset="UTF-8"/>

Essa tag indica o grupo de caracteres que serão usados na pagina para o navegador interpretar, como acentos, alfabeto, etc. Nesse caso, "UTF-8" caracteriza o grupo de caracteres mais comum e que oferece suporte para vários idiomas.


< title>< /title>
