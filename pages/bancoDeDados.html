O que é Banco de Dados?

Bancos de dados são locais/softwares de armazenamento, agrupamento e organização de informações. Os bancos de dados possum um sistema de gerenciamento para facilitar a manipulação e consulta dos dados armazenados. Os sistemas mais comuns são SQL, MySQL, PostgreSQL, Oracle, etc.

HeidiSQL, phpMyAdmin, MySQL Workbench, DBeaver, etc

São "Clientes MySQL", são Softwares utilizados para consultar o software original do banco de dados. Esses softwares são utilizados para fornecer uma interface grafica ao banco de dados de modo a facilitar a visualização e organização.

Operações no MySQL

Porta padrão do MySQL : 3306

C: Create = Criar = Inserir dados 3306

R: Read = Ler = Pegar dados

U: Update = Atualizar = Modificar dados existentes

D: Delete = Excluir = Remover dados existentes

Tipos de dados das colunas

É possivel configurar cada coluna para identificar determinados caracteres, como numeros inteiros, numeros reais, textos, numeros pequenos, numeros extensos, textos extensos, datas, horarios, etc. Veja alguns exemplos:


TINYINT - Para numeros inteiros pequenos ex: 3.

INT - Para numeros inteiros

BIGINT - Para numeros inteiros muito extensos

FLOAT - Para números com virgula ex. valores

VARCHAR - Para textos de até 20 mil caracteres (permitido caracteres especiais)

TEXT - Para textos grandes com paragrafos EX.BLOG

LONGTEXT - Para até 4gb de texto.-

DATE - Para datas (em formato internacional yyyy/mm/dd)

TIME - Para horas

DATETIME - Para data e hora juntas

TIMESTAMP - Para uma conversão de segundos em uma data.

Comandos para consultas

SQL Select


Através desse comando é prossivel fazer uma seleção de informações especificas que queira consultar. Veja exemplos:


SELECT * FROM produtos; - Nesse comando é possivel fazer um levantamento de todas (* = tudo) as informações listadas na tabela produtos


SELECT nome FROM produtos; - Nesse comando é possivel fazer um levantamento de todos os nomes da tabela de produtos


SELECT nome, preco FROM produtos; - Através da virgula é possivel selecionar mais de uma coluna de informações para serem exibidas. Nesse caso, os nomes e os preços.


SQL Where


Através desse comando é possivel especificar e afunilar a consulta, filtrando somente as informações especificas que queira consultar. Veja exemplos:


SELECT * FROM produtos WHERE id_fornecedor = 6; - Selecionar tudo de produtos onde id_fornecedor seja igual a 6.


SELECT * FROM produtos WHERE id = 5; - Selecionar tudo de produtos onde o id seja igual a 5


SELECT * FROM usuarios WHERE nome = "Pedro" ; - Selecionar tudo de usuarios onde o nome for igual a Pedro. ( Aqui é importante o uso das aspas )



SQL Where AND/OR e operadores


SELECT * FROM produtos WHERE preco > 150; - Levantamento dos produtos onde o preço é maior do que 150


SELECT * FROM produtos WHERE preco <= 500; - Levantamento dos produtos onde o preço é menor ou igual a 500


SELECT * FROM produtos WHERE preco != 200; - Levantamento de produtos onde o preço não é igual a 200. ou seja, todos os produtos com excessão dos que custarem exatamente 200.


SELECT * FROM produtos WHERE estoque > minestoque; - Levantamento dos produtos onde o estoque é maior do que o estoque minimo.


SELECT * FROM produtos WHERE preco = 200 OR preco 149; - Levantamento dos produtos em que o preço seja igual a 200 ou quando o preço for igual a 149.


SELECT * FROM produtos WHERE preco > 150 AND preco = 1000 - Levantamento dos produtos onde o preco seja maior do que 150 e que o preco seja igual a 1000.


SELECT * FROM produtos WHERE (preco > 150 AND PRECO < 1000) OR id_fornecedor = 6 Levantamento dos produtos em que o preço é maior do que 150 e que o preço seja menor do que 1000, ou que possuam id_fornecedor - 6 (Aqui é importante o uso de () para separar a ordem das opereções


SQL Where Like


SELECT * FROM usuarios WHERE nome LIKE "Wesley%" ; - Levantamento de usuários que comecem com o nome Wesley


SELECT * FROM usuarios WHERE nome LIKE "%Volpato" ; - Levantamento de usuários que o nome termine com volpato


SELECT * FROM usuarios WHERE nome LIKE "%Volpato%" ; - Levantamento de usuários que possuam "volpato" no nome


SELECT * FROM produtos WHERE nome LIKE "Teclado%" ; - Levantamento dos produtos que comecem com o nome "teclado"


SQL Where Between e IN


SELECT * FROM produtos WHERE estoque BETWEEN 5 AND 10; - Levantamento de produtos em que o estoque esta entre 5 e 10


uma forma alternativa a esse comando seria com a utilização do AND


SELECT * FROM produtos WHERE estoque >= 5 AND estoque <= 10; - Ou seja, produtos que sejam maior ou igual a 5 e que sejam menor ou igual a 10


SELECT * FROM produtos WHERE id_fornecedor IN (1, 6, 3); - Levantamento dos produtos onde o id de fornecedor esteja dentro da seguinte lista (1, 6, 3)


Uma forma alternativa a esse comando seria utilizando o OR


SELECT * FROM produtos WHERE id_fornecedor = 1 OR id_fornecedor = 6 OR id_fornecedor = 3; - Levantamento de produtos em que o id_fornecedor seja 1, 6 ou 3.


SQL Order By


Comando utilizado para ordenar um resultado. É importante que a condição venha primeiro e por ultimo o order by, caso contrario da erro de sintaxe.


SELECT * FROM produtos ORDER BY estoque DSC, nome ASC; - Levantamento dos produtos do estoque por ordem descendente e por ordem alfabetica.


SQL Limit


Esse comando serve para limitar o resultado.


SELECT * FROM produtos LIMIT 5; - Levantamento dos produtos com limite de resultado em 5.


SELECT * FROM produtos ORDER BY preco DESC LIMIT 4; - Levantamento dos preços dos produtos ordenados por ordem decrescente (do mais caro para o mais barato) com limite de visualização de resultados para 4 itens.


SELECT * FROM produtos LIMIT 1, 3 -Levantamento dos produtos pulando um item, começa do segundo item com limite de mostrar 3 resultados. Ex. 10, 12; pula 10 itens e mostra o apenas os 12 itens seguintes.


SELECT * FROM produtos WHERE id_fornecedor = 6 ORDER BY estoque DESC LIMIT 0,3; - Levantamento dos produtos em que o id_fornecedor seja igual a 6, ordenando por estoque de valor maior para o menor, com limite de resultados pulando o 0 e mostrando os 3 seguintes.


SQL Count, Avg e Sum


Esses comandos são utilizados para fazer contagem, média e soma dos resultados. Veja exemplos:


SELECT COUNT(*) FROM produtos WHERE id_fornecedor = 6 ; - Levantamento da CONTAGEM dos produtos onde o id de fornecedor é igual a 6


SELECT AVG(preco) FROM produtos WHERE PRECO >= 1000; - Levantamento da media dos preços dos produtos, onde o preço dos itens é maior ou igual a 1000.


SELECT SUM(estoque) FROM produtos; - Levantamento da soma de produtos do estque
